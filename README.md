# Setup steps
- clone the project
- run `npm install` in the project root
- set your `GEMINI_API_KEY` in `./mock-backend/.env` using `.env.example` as template
- install `expo go` version 2.31.2

# Running the project
## Run Mock backend
- open a terminal window
- enter `cd mock-backend`
- then run `server.js` using `node ./server.js`
## Run the frontend
- open a terminal window
- run `npm start`
